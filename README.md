# VRBowling

졸업 작품으로 진행한 VR 볼링 게임

# 목적

사용자의 볼링 자세를 분석하는 모델을 통해 영상 분석을 VR 게임 환경 속에서 제공하는 모습이다.
초보자의 경우 볼링공의 무게로 인한 부상을 방지하기 위해 VR로 재밌게 볼링 게임을 즐길 수 있다.
